# Mini Chatbot Playground

A simple Gradio based chatbot playground that allows you to chat with multiple open source LLMs (Llama 3, Mistral, Falcon, DeepSeek etc).  
Runs on CPU and can be deployed easily to Hugging Face Spaces.

## Features
- Select different models
- Chat history support
- Clean UI
- Modular design (easy to add more models)
